1. 특정 게시물 댓글 제한 
    - 팔로잉 팔로우 유저만 댓글 가능
    - 팔로우 유저만 댓글 가능
    - 팔로잉 유저만 댓글 가능
    - 댓글이 작성된 상태에서 제한(내가 생각한 기능)
        - 제한된 유저는 본인의 댓글이 보이지만 게시물 작성자 입장에서는 "제한된 댓글" 이라는 문구로 표시됨 
            - 댓글보기로 댓글을 볼 수 있으며 승인/삭제가 가능하다 (승인을 하면 다른사람도 댓글을 볼 수 있다)
            


2. 유저 댓글 제한
    - 나의 모든 게시물에 대한 댓글을 제한할 수 있다. (애초에 댓글을 달 수 없다)
    - 제한하기 전 제한된 사람이 쓴 댓글은 영향을 받지 않는다.
